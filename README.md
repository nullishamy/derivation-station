# NixOS Config

This is my NixOS config, including my home-manager config.

To start, run:
```sh
git clone https://github.com/nullishamy/nix-config/ ~/nixos && cd ~/nixos && nix develop -c "just setup"
```

Once the setup process has finished, rebuild the system.
